React Flashcard App
An interactive, responsive learning tool built to master concepts through active recall and spaced repetition.

Live Demo on Github Pages can be accessed here : https://yeosh235.github.io/react-flashcard-app/  

Project URL can be accessed here : https://roadmap.sh/projects/flash-cards

Tech Stack
Framework: React (Vite)

State Management: React Hooks (useState)

Live link is deployed at Github Pages.  

# Clone and install
git clone https://github.com/yeosh235/react-flashcard-app.git  
cd react-flashcard-app  
npm install  

# Run development server  
npm run dev
